# FileList



![img](https://github.com/jdr2021/java_ListFile/blob/master/1.jpg)

运行该jar包后，输入文件后缀，选择目录。

此时在本目录下会生成一个result.txt文件，该文件中存有指定目录下的直接的所有文件的路径地址。

点击加载，即可渲染到面板上。
